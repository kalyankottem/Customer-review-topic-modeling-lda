# Customer Review Topic Modeling using LDA

## Description
This project uses Latent Dirichlet Allocation (LDA) to uncover hidden themes behind a sudden drop in app ratings by analyzing large volumes of customer reviews.

The system identifies major complaint topics, extracts representative keywords, and generates business recommendations to improve customer satisfaction.

---

## Problem Statement
A company’s app rating has dropped significantly due to rising customer dissatisfaction. Manual analysis of thousands of reviews is impractical.

This project automates root-cause discovery using topic modeling.

---

## Objective
- Analyze large-scale customer review data  
- Discover hidden complaint themes using LDA  
- Extract top keywords per topic  
- Interpret topics into business issues  
- Recommend countermeasures  

---

## Dataset
Synthetic dataset of 2000 customer reviews simulating real-world app complaints.

---

## Methodology
1. Generate / Collect review dataset  
2. Preprocess review text  
3. Convert text into numerical form  
4. Apply LDA Topic Modeling  
5. Extract topic keywords  
6. Analyze topic frequency distribution  
7. Generate business recommendations  

---

## Output
The project generates:
